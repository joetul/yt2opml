# yt2opml

Convert your YouTube subscriptions into an OPML file you can import into any RSS reader (like FeedFlow, Feeder, NetNewsWire, etc).

**Try it here:** https://joetul.github.io/yt2opml/

Everything runs in your browser. No data is sent anywhere.

## How to use

1. Export your YouTube subscriptions from [Google Takeout](https://takeout.google.com/)
   - Deselect all, then select only **YouTube and YouTube Music**
   - Click "All YouTube data included" and pick only **subscriptions**
   - Create the export and download the zip
2. Open the website and drop your `subscriptions.csv` file on it
3. Click **Convert & Download OPML**
4. Import the downloaded `.opml` file into your RSS reader

## Run locally

Download or clone this repo and open `index.html` in your browser.
